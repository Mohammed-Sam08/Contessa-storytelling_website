# Contessa-storytelling_website
Hindustan Contessa | The Indian Muscle
A premium, interactive scrollytelling experience dedicated to the legacy of the Hindustan Contessa. This project uses high-performance web technologies to deliver a cinematic story of "The Indian Muscle" through a dark, modern "Deep Noir" aesthetic.

🌟 Key Features
Interactive Scrollytelling Engine: A custom-built animation system using HTML5 Canvas and React to render smooth, frame-by-frame automotive sequences synced with user scroll.
Cinematic Narrative Overlays: Floating textual overlays with parallax and opacity transforms that reveal the story of the Contessa as the user explores.
"Deep Noir" Aesthetic: A meticulously crafted UI using a dark, premium color palette (#080808) with accent ambers and silvers for a luxury feel.
Responsive Design: Fully optimized for various screen sizes, ensuring the premium experience is maintained across desktop and mobile devices.
High-Performance Animations: Powered by Framer Motion for sophisticated UI transitions and micro-animations.
Modern Landing Page Sections: Includes features/specifications grid, a legacy-themed about section, and an integrated newsletter registration.
🛠️ Technology Stack
Framework: Next.js 14 (App Router)
Language: TypeScript
Animation: Framer Motion
Styling: Tailwind CSS
Rendering: HTML5 Canvas (for frame-based sequences)
Deployment Ready: Optimized for Vercel or any modern Node.js environment.
🚀 Getting Started
Clone the repository
Install dependencies:
bash
npm install
Run the development server:
bash
npm run dev
Build for production:
bash
npm run build
📸 Project Structure
/app: Next.js App Router components and layouts.
/public: Static assets including high-resolution car frames and branding media.
ContessaScroll.tsx: The core animation engine managing scroll-synced frame rendering.
